Modeling low-code databases with executable UML

With this project, you're able to generate a database schema in Baserow from the UML class diagram. It's essentially a REST API for transforming UML class diagram into Baserow database. UML class diagram as an XMI file is parsed in Python and then a database is created with all tables as classes, fields as attributes, enumerations and associations from a model.
